# Quarto

This blog is created with [Quarto](https://quarto.org/) an open source scientific publishing system. 

## Preview 

- Full Website: ```quarto preview```
- Drafting/Specific File: ```quarto preview --profile draft {qmd filename}```

## Publish 

Ensure the repo has the [proper structure](https://quarto.org/docs/publishing/github-pages.html)

quarto publish gh-pages
